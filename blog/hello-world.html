Hello, World! — 我的第一篇博客

2026-07-05 · 3 分钟阅读

一直以来都有写博客的想法，但总是「下次一定」。今天终于借着 GitHub Pages 把这个坑填上了。

为什么开始写博客

写博客的目的很简单：

  • 整理知识 — 教是最好的学，写出来才能真正理解
  • 记录成长 — 回头看以前的文章，能看到自己的变化
  • 分享交流 — 也许我的经验能给其他人一些帮助

搭建过程

这个博客是用纯静态 HTML + CSS + JS 搭建的，部署在 GitHub Pages 上。整个过程非常简单：

  1. 创建一个名为 starfading.github.io 的仓库
  2. 把本地站点文件 push 到仓库的 main 分支
  3. 几分钟后就可以通过 https://starfading.github.io 访问了

GitHub Pages 的优点在于完全免费、支持自定义域名、与 Git 工作流无缝集成。

未来计划

后续我会在这里分享更多技术文章，包括前端开发、系统设计等方面的学习笔记和实践心得。也会记录一些生活中有趣的见闻。

如果你也有写博客的想法，不妨现在就动手试试吧。
